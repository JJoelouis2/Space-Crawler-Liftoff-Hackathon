# Cry Baby's Garden
The team read a book called the "Three Body Problem" and decided it was sad that the Trisolaran aliens were stuck in a
hostile environment. Plus, the Trisolaran aliens were void of any emotions. Despite the Trisolaran's goal of eradicating humanity
and stealing Earth, we decided a nice game that would fix their issues might change their mind. This is that nice game.

# Features
- Horrific, eary atmosphere
- Occasional jump scares
- Calm gardening experience
